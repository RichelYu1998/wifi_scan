## 🔧 预算范围算法

### 算法逻辑
```python
def calculate_budget_range(budget):
    """计算预算范围"""
    if budget < 1000:
        # 低预算：±500元，最低500元
        min_budget = max(500, budget - 500)
        max_budget = budget + 500
    elif budget < 3000:
        # 中低预算：±500元
        min_budget = budget - 500
        max_budget = budget + 500
    elif budget < 8000:
        # 中等预算：±1000元
        min_budget = budget - 1000
        max_budget = budget + 1000
    else:
        # 高预算：±2000元
        min_budget = budget - 2000
        max_budget = budget + 2000
    
    return min_budget, max_budget
```

### 算法特点
1. **分层处理**：根据预算大小采用不同的范围策略
2. **最低保护**：低预算时设置最低500元的保护线
3. **合理范围**：确保预算范围既不过宽也不过窄
4. **边界平滑**：边界条件处理平滑，避免跳变
