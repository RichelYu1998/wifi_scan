# WiFi扫描与投影仪推荐系统 - 集成版

## 📋 概述

`integrated_system.py` 是系统的核心集成模块，整合了所有功能到一个统一的用户界面中。

## 🎯 主要功能

### 1. 硬件信息检测
- CPU信息检测（型号、核心数、频率）
- GPU信息检测（品牌、型号、显存）
- 内存信息检测（容量、频率、DDR类型）
- 硬盘信息检测（容量、读写速度）
- BIOS版本检测和最新版本对比
- 主板信息检测（品牌、型号）

### 2. WiFi网络扫描
- 扫描周围WiFi网络
- 显示网络详细信息（信号强度、加密方式、频道等）
- 支持网络连接测试

### 3. 投影仪推荐系统
- **按预算推荐**：智能预算范围计算，支持国补价格对比
- **按品牌搜索**：支持32个主流投影仪品牌搜索
- **联网搜索**：模拟电商API搜索最新投影仪信息
- **多维度性价比评估**：价格、显示性能、智能功能、使用体验、能效环保
- **详细商品信息**：显示完整的商品技术参数
- **京东链接跳转**：直接跳转到京东商品页面

### 4. 网络速度测试
- 上传速度测试
- 下载速度测试
- 网络延迟测试

### 5. 运行所有功能
- 一键运行所有功能模块
- 生成综合测试报告

## 🐛 故障排除

### 常见问题

#### 1. Python版本不兼容
**问题**：显示"Python版本过低"错误
**解决**：升级到Python 3.7+

#### 2. 依赖库缺失
**问题**：显示"ModuleNotFoundError"
**解决**：运行`pip install psutil requests`

#### 3. 中文显示乱码
**问题**：中文显示为乱码
**解决**：使用`start.bat`启动，它会设置UTF-8编码

#### 4. 虚拟环境未检测
**问题**：系统提示"未检测到虚拟环境"
**解决**：确保虚拟环境目录名为`.venv`、`venv`或`env`

#### 5. 京东链接无法打开
**问题**：点击京东链接后显示错误页面
**解决**：使用智能搜索链接，系统会自动优化搜索参数
